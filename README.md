# CCONECT-DATABASE
Complete ERD and SQL Server database schema for the Final Year Project, including DDL scripts, constraints, and sample data for database implementation and practice.

## Database Seeding
- Categories and services are system-seeded
- User-related data is inserted through the application UI
- Seed data supports service discovery and booking flows

## Role & Responsibilities (This Repository)

This project was developed as a team-based Final Year Project.  
This repository contains the **database module**, which was designed and implemented by me.

My responsibilities included:
- Designing the relational database schema using SQL Server
- Creating ERD diagrams and applying normalization principles
- Defining tables, primary keys, foreign keys, and constraints
- Writing complete SQL DDL scripts
- Preparing seed data for categories and services
- Ensuring data integrity, consistency, and referential integrity

Frontend and backend implementation details are documented in their respective repositories.
